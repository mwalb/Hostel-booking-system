# Hostel Booking System

A simple Django web application for managing hostel room bookings.

## Features
- Add new bookings
- View all bookings
- Update booking details
- Delete bookings
- Basic form validation

## Technologies Used
- Django (Python)
- HTML, bootstrapCSS
- SQLite Database

## How to Run
1. Clone the repository  
2. Install requirements  
3. Run migrations  
4. Start the server  
5. Open in browser: http://127.0.0.1:8000/
